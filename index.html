⚙ Setup Discord login

Skip the .env file — paste your Discord App's Client ID and Client Secret here. They live in this server's memory only (not on disk).

  1. Open the Discord Developer Portal → New Application.
  2. Go to OAuth2 → General. Add this Redirect URL: http://localhost:3000/auth/callback
  3. Copy the Client ID and Client Secret (under "Reset Secret") and paste below.
  4. Bot tab → Add Bot → enable Server Members Intent + Message Content Intent → copy the Bot Token.
  5. OAuth2 → URL Generator → scope bot → permissions View Channels, Read Message History, Send Messages → open the URL → invite the bot to your server.
